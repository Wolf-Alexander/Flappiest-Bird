# Flappiest Bird

This is a simple version of a Flappy Bird game made with Unity

## Important Notes

This project is based on a tutorial. The purpose of this project is for me to learn the basics of Unity. The main structure and the assets have been taken from this tutorial:

https://www.youtube.com/watch?v=XtQMytORBmM

There are still features that are made by me:
- Starting Screen
- A lot of Error handlings
- Important mechanics for smooth gameplay

## Installation

Simply downlaod the latest release and run the .exe file (It's probably called "Flappiest_Bird_vx.x.x)

## Features

This game has the main Flappy Bird features you would expect.

- Bird Controlling
- Pipe Spawning
- Borders checking
- Game Over handling
