# Flappiest Bird

This is a simple version of a Flappy Bird game made with Unity

## Important Notes

This project is based on a tutorial. The main structure and the assets have been taken from this tutorial:

https://www.youtube.com/watch?v=XtQMytORBmM


## About This Project
 
This project started from a tutorial, which provided the base structure and initial assets. My goal was to learn Unity fundamentals by building on top of it.
 
**What I added on top of the tutorial:**
- A custom start screen and menu flow
- Difficulty selection system
- High score persistence
- Background music and sound effect integration
- Extensive error handling for edge cases (e.g. rapid input, boundary collisions)
- General gameplay polish for smoother, more consistent feel

## Features

This game has the main Flappy Bird features you would expect.


- Bird Controlling
- Pipe Spawning
- Borders checking
- Game Over handling
